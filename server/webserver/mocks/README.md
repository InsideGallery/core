# server/webserver/mocks

`mocks` contains generated GoMock test doubles for `server/webserver`.

## Main APIs

- `NewMockHTTPClient(ctrl)`: creates a generated mock for the legacy
  `webserver.HTTPClient` interface.
- `MockHTTPClient.EXPECT().Do(req)`: configures expectations for outbound
  `Do(*http.Request)` calls.

## Usage

```go
ctrl := gomock.NewController(t)
client := mock_webserver.NewMockHTTPClient(ctrl)

client.EXPECT().Do(gomock.Any()).Return(&http.Response{
	StatusCode: http.StatusOK,
	Body:       io.NopCloser(strings.NewReader("ok")),
}, nil)
```
