# Source Reference Materials

Engineering reference documents for AI agents and developers working on Ptolemy.

## Committed Files (tracked in git)

| File | Description |
|------|-------------|
| `Go Server.md` | **Primary architecture reference.** MDCA (Modular Domain-Centric Architecture), DDD folder structure, Model/DTO separation rules, single-domain vs multi-domain patterns, event-driven design, Go best practices. |
| `AIC Template.md` | Architecture Inception Canvas template for creating new epics in `docs/aics/`. |

## Local-Only (gitignored)

This folder also supports local-only reference docs that are not committed. Each developer can add their own. Examples: `Clean Code.md`, `Go Best Practice.md`, `Twelve-Factor App.md`, etc.
